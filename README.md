# TP_1_SO
Repository for the first practical project of the Operating Systems course - UFMG 2021/1

Compile with :

`gcc sh.c -o myshell.out`

Execute the shell with : 

`./myshell.out`

This shell only supports simple commands such as : 

`cd dir`

`ls > y`

`cat < y | sort | uniq | wc > y1`

`cat y1`

`rm y1`

`ls | sort | uniq | wc`

`rm y`
